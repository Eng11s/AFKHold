<p style="text-align:center"><b>This mod was made to simulate taping down buttons while AFK. This can be very useful when using any farm that require holding buttons down, clicking or making tedious tasks like walking/boating long distances simple.</b></p>
<br>
  

## Introduction

All you need to do is press '**K**' (or you can bind it somewhere else) to keep them held down. Once held down, it frees your mouse and you can do something else.  
If you need to use mouse while held down, you can hold '**LALT**' to look around in game.

**Mod Settings**

*   Mute game while held down
*   Disconnect on damage
*   Disconnect on totem use
- Autoclicker switch
- Autoclicker delay

## Requirements

Optional

*   [Mod Menu](https://www.curseforge.com/minecraft/mc-mods/modmenu)

## Key Bindings

| Name       |Default Key |Description                                        |
| ---------- |----------- |-------------------------------------------------- |
| <code>Activate</code> |<code>K</code> |Activates holding down selected keys               |
| <code>Grab Mouse</code> |<code>Left Alt</code> |Grabs mouse input while actively holding keys down |
