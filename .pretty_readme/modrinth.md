## Introduction
All you need to do is press '**K**' (or you can bind it somewhere else) to keep them held down. Once held down, it frees your mouse and you can do something else.<br>
If you need to use mouse while held down, you can hold '**LALT**' to look around in game.

**Mod Settings**
- Mute game while held down
- Disconnect on damage
- Disconnect on totem use
- Autoclicker switch
- Autoclicker delay

## Requirements
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [YACL](https://modrinth.com/mod/yacl)

Optional
- [Mod Menu](https://modrinth.com/mod/modmenu)

## Key Bindings
| Name         | Default Key | Description                                        |
|--------------|-------------|----------------------------------------------------|
| `Activate`   | `K`         | Activates holding down selected keys               |
| `Grab Mouse` | `Left Alt`  | Grabs mouse input while actively holding keys down |
